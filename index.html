    Onrender

    How to make links with Render. Requirements: A github account Sometimes It will ask for a debit/credit card for verification, this will be for a very few amount of people. Visit our github and then fork the repository Github Link How to deploy Signup/Login to OnRender

  • Click "New +"
  • Click "Web Service"
  • Connect your github account.
  • Choose the interstellar repo.
  • Name the Project
  • Scroll down and click "Create Web Service" at the bottom of the page.
  • Then wait for a minute or two and it will be done deploying.
  • If it says "Certificate Pending" refresh the page, and if it doesn't go away wait for it to go away
